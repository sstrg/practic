# Отчёт по учебной практике

Данный репозиторий содержит материалы учебной практики, направленной на изучение способов организации данных, алгоритмов их обработки и применения в задачах программирования на языке Python.

Практика состоит из **16 заданий**, каждое из которых оформлено в виде отдельной директории с исходным кодом.

---

## Как открыть и запустить задания

1. Склонируйте репозиторий или скачайте его в виде архива и распакуйте на компьютер.

2. Откройте папку проекта `practic` любым удобным способом:
   - через файловый менеджер
   - через терминал
   - через среду разработки (PyCharm, VS Code и др.)

3. Каждое задание находится в отдельной директории.

4. Внутри каждой директории расположен файл `N.py`, где 'N' - название задания, содержащий решение соответствующего задания.
