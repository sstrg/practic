# Отчёт по учебной практике

Данный репозиторий содержит материалы учебной практики, направленной на изучение способов организации данных, алгоритмов их обработки и применения в задачах программирования на языке Python.

Практика состоит из **16 заданий**, каждое из которых оформлено в виде отдельной директории с исходным кодом.
